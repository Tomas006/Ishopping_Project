## Descrição

Aplicação desktop em C# WinForms para gestão de compras domésticas.
Permite gerir utilizadores, tipos de artigo, artigos, orçamentos mensais,
planear listas de compras, realizar compras com controlo de orçamento em
tempo real, exportar dados para CSV e consultar estatísticas e sugestões
de orçamento e lista de compras.

---

## Requisitos

- .NET Framework 4.7.2
- SQL Server LocalDB
- Connection string configurada no `App.config` com o nome `IShoppingContext`
